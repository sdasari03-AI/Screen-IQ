EXTENSION 1 — AI INTELLIGENCE PANEL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Add a new "AI Intelligence" tab to the 
existing screening run detail page.

Five sub-panels inside this tab:

CHARGE CLASSIFIER
- Input: raw criminal charge string
- GPT-4o returns: category 
  (Property/Violent/Drug/Traffic/
  White Collar/Other), severity 
  (Felony/Misdemeanor/Infraction), 
  disposition (Convicted/Dismissed/
  Acquitted/Pending), one-sentence 
  plain language summary
- Employer ruleset: dropdown to select 
  "Flag felonies only" or 
  "Ignore traffic violations" — 
  AI applies ruleset automatically
- Output badge: Green/Yellow/Red

CHARGE EXPLAINER
- One-click button on any flagged charge
- GPT-4o returns: plain language meaning, 
  typical sentence range, relevant statute, 
  how common this charge is, context for 
  fair hiring decision
- Displayed as a readable card

NAME MATCHER
- For every candidate, AI identifies 
  common aliases, married/maiden variants, 
  hyphenated combinations
- Confidence score per variant (0-100%)
- Shows: how many variants searched, 
  how many returned records, 
  any new matches found under aliases

ROLE MATCHER
- Candidate's claimed job title and 
  employer vs verification record
- GPT-4o flags: Confirmed / Discrepancy 
  / Fabrication / Unable to verify
- Shows exact mismatch text:
  "Claimed: Senior Engineer — 
   Record shows: Engineer II"

DOC INSPECTOR
- Upload zone for pay stubs, bank 
  statements, offer letters
- GPT-4o Vision scans for: font 
  inconsistencies, alignment issues, 
  metadata mismatches, altered fields
- Output: Authentic / Review Required 
  / Likely Altered
- Shows specific flags found

EXTENSION 5 — DRUG TESTING MODULE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Add Drug Testing as a check type 
inside the existing screening package 
creation flow.

Test types:
- 5-Panel Urine (standard non-DOT)
- 10-Panel Urine (extended)
- DOT 5-Panel (federally regulated)
- Hair Follicle (90-day window)
- Oral Fluid (recent use)

Workflow inside existing screening run:
1. Drug test ordered with package
2. Candidate sees collection site 
   locator (nearest sites listed)
3. Collection completed
4. Lab processes specimen
5. MRO review for non-negative results
6. Result: Negative / Positive / 
   Dilute / Refused / Invalid / Cancelled

DOT compliance flags:
- DOT tests blocked from finalizing 
  without MRO review
- Chain of custody tracked per test
- Non-negative results held in 
  "Pending MRO Review" status
